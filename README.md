# Graph Theory Labs (SPbSTU, 2nd year 2nd sem., Spring 2025/2026)

This repository contains laboratory works for the Graph Theory course at Peter the Great St. Petersburg Polytechnic University.

## 📚 Contents

### Lab 1
- Random graph generation based on probability distributions
- Graph properties:
  - Eccentricity
  - Diameter
  - Center
- Shimbell method (min/max paths)
- Route existence and counting

### Lab 2
- Graph traversal:
  - BFS (Breadth-First Search)
  - DFS (Depth-First Search)
- Graph algorithms:
  - Dijkstra
  - Bellman-Ford
  - Floyd-Warshall
- Performance comparison

### Lab 3
- Flow networks:
  - Capacity and cost matrices
- Maximum flow:
  - Ford-Fulkerson algorithm
- Minimum-cost flow
